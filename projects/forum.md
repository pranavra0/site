I was inspired by old forums and particularly how old websites managed state. I decided to use Go to create a mostly server side forum. This was also my first time using HTML Templating which significantly changed how I view web development. 

### Features
* Accounts: Simple auth
* Simple but robust backend 
* CLI based moderation tools
* Replies: Nested threading that's actually good for conversation flow
